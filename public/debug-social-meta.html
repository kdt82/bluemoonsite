‚úÖ Social Media Meta Tags Checklist

üì± Essential Open Graph Tags

og:title - Main title for social sharing
og:description - Description text (155-160 characters ideal)
og:image - Image URL (1200x630px recommended)
og:url - Canonical URL of the page
og:type - Content type (article, website, etc.)
og:site_name - Name of your website

üê¶ Essential Twitter Card Tags

twitter:card - Card type (summary_large_image recommended)
twitter:title - Title for Twitter (70 characters max)
twitter:description - Description for Twitter (200 characters max)
twitter:image - Image URL (same as og:image)
twitter:site - Your Twitter handle (optional)

üì∞ Article-Specific Tags

article:published_time - Publication date
article:author - Author name
article:section - Content category
article:tag - Keywords/tags

üìã Meta Tags Implementation Guide

Next.js Layout.tsx Format:

export const metadata: Metadata = {
  metadataBase: new URL('https://yourdomain.com'),
  title: 'Your Page Title | Site Name',
  description: 'Your page description',
  openGraph: {
    title: 'Your Page Title',
    description: 'Your page description',
    images: ['/assets/your-image.png'],
    type: 'article',
    url: '/your-page-url',
  },
  twitter: {
    card: 'summary_large_image',
    title: 'Your Page Title',
    description: 'Your page description',
    images: ['/assets/your-image.png'],
  },
};

HTML Meta Tags Format:

<meta property="og:title" content="Your Page Title" />
<meta property="og:description" content="Your page description" />
<meta property="og:image" content="https://yourdomain.com/assets/your-image.png" />
<meta property="og:url" content="https://yourdomain.com/your-page" />
<meta property="og:type" content="article" />

<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Your Page Title" />
<meta name="twitter:description" content="Your page description" />
<meta name="twitter:image" content="https://yourdomain.com/assets/your-image.png" />

üîß Common Issues & Solutions

üñºÔ∏è Image Not Displaying

  • Use absolute URLs: Always use full https:// URLs for images
  • Check image size: 1200x630px is optimal (1.91:1 aspect ratio)
  • File size limit: Keep under 5MB for Facebook/Twitter
  • Format: Use PNG or JPG formats
  • Public access: Ensure image URL is publicly accessible

üîÑ Cache Issues

  • Facebook: Use Facebook Debugger and click "Scrape Again"
  • Twitter: Cards are usually updated within a few minutes
  • LinkedIn: Use Post Inspector to refresh
  • WhatsApp: May take 24-48 hours to update

üìù Content Issues

  • Title too long: Keep Facebook titles under 100 characters
  • Description length: 155-160 characters ideal for most platforms
  • Missing tags: Ensure all essential tags are present
  • Duplicate content: Each page should have unique meta tags

üîç Validation Steps

  1. Check page source for meta tags
  2. Validate with Facebook Debugger
  3. Test with Twitter Card Validator
  4. Verify image loads independently
  5. Test actual sharing on platforms
